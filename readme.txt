Ein schlankes Wartungsmodus/Coming Soon Plugin (Baker Street).

= Features =
* Wartungsseite (responsive) für nicht eingeloggte Besucher
* Eingeloggte Benutzer sehen die Website normal
* Headline, Text, Telefon, E-Mail, Logo, Schrift, Farben im Backend konfigurierbar
* /wartungsmodus wird im Webroot angelegt und bleibt auch bei deaktiviertem Plugin erreichbar
* noindex via Meta + X-Robots-Tag

= Bedrock / Composer =
Das Plugin enthält eine composer.json mit "type": "wordpress-plugin".
Du kannst es z.B. via VCS-Repository in dein Bedrock-Projekt einbinden.
